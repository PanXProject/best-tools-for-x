## About
Datasette lets you host and query datasets on your server. Upload CSV files and share interactive views with your team. Perfect for analysts or small orgs publishing data without heavy tools.

## Features
- Upload CSV for instant querying
- Generate shareable data URLs
- Self-host with SQLite backend
- Filter data with SQL commands
- Export results as JSON or CSV
