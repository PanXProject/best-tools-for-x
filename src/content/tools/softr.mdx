## About
Softr turns Airtable data into web apps with a free plan for one app. Use it to create client portals or tools without coding. Great for entrepreneurs or small teams leveraging Airtable.

## Features
- Build one app from Airtable
- Support up to 1000 rows
- Add 5 editors for collaboration
- Customize with drag-and-drop
- Publish live instantly
