## About
Studiovity is an AI-powered script breakdown software that automatically extracts production elements from screenplays, including characters, props, locations, costumes, vehicles, VFX, and more. Generate breakdown reports and connect them directly to scheduling, budgeting, call sheets, and production workflows.

## Features
- AI Script Element Extraction
- Automatic Scene Tagging
- Cast, Props & Location Breakdown
- Production Breakdown Reports
- Schedule & Budget Integration
