## About
MagicForm uses AI to generate quizzes from any content source including PDFs, URLs, YouTube videos, or text. Ideal for educators and trainers who need quick assessment creation without manual question writing.

## Features
- Generate quizzes from any content source
- AI-powered question creation
- Multiple question types support
- Export to various formats
- Customizable quiz settings
- Instant quiz generation
- Google Forms integration
- Bulk quiz creation
- Answer key generation
- Performance analytics
