## About
Zammad is a free, self-hosted helpdesk for managing customer support tickets. Use it to handle inquiries via email or web without subscription costs. Perfect for small teams needing open-source support tools.

## Features
- Create unlimited tickets
- Manage via email or web interface
- Self-host for full control
- Assign tickets to team members
- Track resolution times
