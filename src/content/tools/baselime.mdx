## About
Baselime monitors serverless apps for errors and performance. Use it to debug Lambda functions with real-time logs. Ideal for devs running lean, distributed systems.

## Features
- Monitor serverless logs
- Trace errors in real time
- Set custom alerts
- Analyze performance metrics
- Integrate with AWS
