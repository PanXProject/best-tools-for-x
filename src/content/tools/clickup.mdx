## About
ClickUp manages tasks, docs, and goals in one platform. Use it to track team projects with custom views. Great for replacing multiple tools with a single, flexible hub.

## Features
- Create tasks with deadlines
- Switch between list and board views
- Store docs with tasks
- Track time on projects
- Assign tasks to team
