## About
Debuggix scans your code with 9 security engines in 60 seconds. 
Find SQL injection, hardcoded secrets, dependency CVEs, and more. 
Built for indie devs and vibe coders who ship fast and skip security. 
Free tier available. No credit card required.

## Features
- 9 security engines in one scan
- AI-generated fixes for every vulnerability
- Scans code, dependencies, Dockerfiles, and infrastructure-as-code
- GitHub integration with automatic PR creation
- Live VS Code panel with real-time security linting
- Team collaboration with role-based access
- Zero data retention — your code stays yours
- Free tier with 10 scans/month
- 60-second full scan for repos up to 10,000 lines
- Slack integration for real-time vulnerability alerts
- Custom security rules engine (Pro+)
- Webhook support for CI/CD pipelines
- Executive summary reports for stakeholders
- Compliance readiness (SOC2, ISO, GDPR)

- Bulk scan support via API
- Scan scheduling for automated checks
