## About
Snyk scans your code for vulnerable dependencies and suggests fixes. Use it in your CI pipeline to catch issues early. Ideal for devs building secure apps without slowing down.

## Features
- Scan repos for vulnerabilities
- Suggest fixes via CLI
- Integrate with GitHub
- Monitor dependencies daily
- Test open-source packages
