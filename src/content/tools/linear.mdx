## About
Linear tracks bugs and features for dev teams with a fast, focused UI. Use it to manage sprints without bloated tools. Ideal for small software teams prioritizing speed and clarity.

## Features
- Create issues with shortcuts
- Assign tasks to team members
- Track progress in sprints
- Sync with GitHub commits
- Filter issues by priority
