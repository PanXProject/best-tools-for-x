## About
Kopia backs up files to cloud or local storage with encryption. Use it to protect critical data on your desktop securely. Great for individuals or small teams needing fast restores.

## Features
- Encrypt backups end-to-end
- Deduplicate files for space
- Restore files via CLI
- Backup to S3 or local drives
- Schedule daily snapshots
