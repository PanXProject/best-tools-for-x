## About
Bonsai streamlines invoicing and contracts for freelancers. Use it to send professional bills and sign deals fast. Ideal for solo workers managing clients without admin overload.

## Features
- Send invoices in one click
- Create e-signable contracts
- Track payment status
- Automate late payment reminders
- Manage clients in dashboard
