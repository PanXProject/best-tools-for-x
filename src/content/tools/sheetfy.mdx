## About
Sheetfy directly connects your Google Sheets and Google Drive spreadsheets and transforms them into complete REST APIs. Save days by avoiding the creation and maintenance of a backend.

## Features
- Instantly transform any Google Sheet into a REST API
- Access spreadsheet data using standard HTTP requests (GET, POST, PUT, DELETE)
- Control access with API keys and permissions
- Log and monitor API usage and requests
- Keep your data always in sync with Google Sheets, without manual updates
