## About

Neon is a serverless PostgreSQL database that separates storage and compute for instant scaling. Create database branches like Git repositories for development workflows. Pay only for what you use with automatic scaling to zero.

## Features

- Serverless PostgreSQL database
- Instant database branching
- Automatic scaling to zero
- Pay-per-use pricing model
- Point-in-time recovery
- Read replicas
- Connection pooling
- CLI and web interface
- GitHub integration
- Developer-friendly APIs
