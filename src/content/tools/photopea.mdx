## About
Photopea edits images in the browser like Photoshop, for free. Use it to tweak graphics without installing software. Perfect for quick edits or users on low-spec devices.

## Features
- Edit PSD files online
- Use layers and masks
- Export in multiple formats
- Work without signup
- Access advanced filters
