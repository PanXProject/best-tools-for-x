## About
TinaCMS is an open-source content management system that enables real-time editing of markdown content, fully backed by Git. Developers can seamlessly integrate CMS features directly into their projects, empowering teams to collaborate, manage, and publish content efficiently across websites and digital platforms.

## Features
- Real-time editing of markdown and JSON content
- Git-backed version control for all changes
- Seamless integration into modern web projects
- Customizable editorial workflows
- Collaborative content management
- Easy setup for static and dynamic sites
- Extensible with custom plugins
- Live preview of content updates
- User roles and permissions
- Content scheduling and publishing
