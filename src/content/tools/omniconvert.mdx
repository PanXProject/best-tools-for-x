## About
Omniconvert runs A/B tests and personalizes web experiences for better conversions. Use it to tweak pages based on user behavior without coding. Great for e-commerce or marketing teams optimizing funnels.

## Features
- Set up A/B tests with a visual editor
- Personalize content for segments
- Track conversion goals
- Integrate with analytics tools
- Analyze test results in real time
