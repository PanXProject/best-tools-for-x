## About
Cabin offers privacy-first web analytics that respects user privacy while providing essential website insights. No cookies, no personal data collection, just clean metrics you need to improve your site.

## Features
- Cookieless tracking
- Privacy-compliant analytics
- Real-time visitor data
- Page performance metrics
- Referral source tracking
- Goal and event tracking
- Simple dashboard interface
- GDPR compliant by design
