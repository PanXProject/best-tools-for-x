## About
AND.CO helps freelancers manage one client for free with invoicing and contracts. Use it to send professional invoices and agreements without fees. Best for solo freelancers starting out.

## Features
- Create invoices for one client
- Draft and send contracts
- Track time on projects
- Accept payments online
- Access basic proposal templates
