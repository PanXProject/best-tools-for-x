## About
Slab stores team docs and processes in a searchable wiki. Use it to onboard new hires or keep SOPs accessible. Perfect for remote teams needing a single source of truth.

## Features
- Search docs instantly
- Organize with nested pages
- Edit collaboratively in real time
- Integrate with Slack
- Track version history
