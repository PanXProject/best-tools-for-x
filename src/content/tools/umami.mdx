## About
Umami tracks site stats without cookies, self-hosted for privacy. Use it to monitor traffic simply without GDPR hassles. Great for indie devs or small sites wanting lightweight analytics.

## Features
- Add tracking with one script
- View real-time visitors
- Self-host with Docker
- See top pages and referrers
- Export data easily
