## About

Firecrawl is an API-first crawling service that returns cleaned, structured content from websites. It is useful for teams that need consistent page-level text and metadata for model training, search indexes, or automated monitoring.

## Features

- Site-wide crawl with per-page markdown/JSON output.
- Boilerplate removal and text normalization for LLM inputs.
- API for programmatic control and scheduling.
- Screenshot capture alongside extracted text.
- Designed to scale for large sites and frequent crawls.
