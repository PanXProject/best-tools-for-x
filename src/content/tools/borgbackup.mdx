## About
BorgBackup saves space with deduplication and encrypts backups via CLI. Use it to archive server data efficiently. Great for tech users needing lightweight, secure backups.

## Features
- Deduplicate files automatically
- Encrypt with strong keys
- Backup to remote servers
- Mount backups as filesystems
- Prune old backups easily
