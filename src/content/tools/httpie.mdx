## About

HTTPie is a user-friendly HTTP client designed for testing, debugging, and interacting with APIs and HTTP servers. It offers an intuitive command-line interface and web application that makes HTTP requests human-readable and easy to work with.

## Features

- Expressive and intuitive syntax
- Built-in JSON support with syntax highlighting
- Support for REST and GraphQL APIs
- Persistent sessions and authentication
- Cross-platform CLI tool
- Web-based interface for browser testing
- Custom headers and request methods
- Beautiful output formatting
