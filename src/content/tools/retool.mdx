## About
Retool lets you build custom internal apps by connecting to databases. Create a dashboard for your team in hours, not weeks. Great for ops teams needing tailored tools without dev resources.

- Drag-and-drop UI builder
- Write custom JavaScript logic
- Share tools with team links
- Export data to CSV
