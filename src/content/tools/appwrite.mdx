## About
Appwrite gives developers a self-hosted backend for apps, handling auth and storage. Set it up on your server to avoid cloud vendor costs. Ideal for custom app projects with full control.

## Features
- Add user auth with one API
- Store files securely on your server
- Manage databases via dashboard
- Scale with Docker integration
- Open-source with no licensing fees
