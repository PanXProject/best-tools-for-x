## About
React Bits is a collection of high-quality, animated, and customizable React components. It's a great resource for developers looking to add unique and interactive elements to their web projects without building them from scratch. This tool helps you create stunning user interfaces and provides the code for each component, making it easy to integrate and modify to fit your needs.

## Features
- 100+ components for animation and UI
- Code is fully customizable
- Components are interactive
- Open-source and free to use
- Available in JS, TS, CSS, and Tailwind variants
