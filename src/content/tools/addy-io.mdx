## About
addy.io protects your real email by generating unlimited aliases that forward to your inbox. Each service gets a unique address, preventing spam and tracking. Open-source solution with anonymous reply support.

## Features
- Create unlimited email aliases for free
- Anonymous replies to protect identity
- Self-hosting option for full control
- Browser extension for quick generation
- Multiple domain options available
- PGP encryption support
- Detailed forwarding statistics
- API access for automation
- Custom recipient management
- Advanced filtering options
- Mobile app support
- Backup and restore options
- Multi-language interface
- Custom domain integration
- Rate limiting protection
- Webhook notifications
