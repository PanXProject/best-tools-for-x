## About
Nifty simplifies project management with task lists and kanban boards. Ideal for marketing or HR teams, it supports small teams for free. Its clean interface is easier to use than complex tools like Asana, helping teams stay organized.

## Features
- Create task lists
- Use kanban boards
- Track time on tasks
- Share files
- Set milestones
- Chat with team
- Plan with Gantt charts
- Customize workflows
- Access on mobile
- View analytics
