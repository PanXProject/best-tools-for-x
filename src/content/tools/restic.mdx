## About
Restic creates encrypted backups via CLI for tech-savvy users. Use it to save files to remote servers securely. Perfect for sysadmins managing data with command-line precision.

## Features
- Encrypt backups with AES
- Deduplicate to save space
- Backup to cloud providers
- Restore specific files
- Verify backup integrity
