## About
Caddy serves sites with automatic HTTPS, no config hassles. Use it to host static files or apps securely. Great for devs needing a lightweight, modern web server.

## Features
- Enable HTTPS by default
- Serve static files fast
- Self-host with one binary
- Reverse proxy to apps
- Customize with Caddyfile
