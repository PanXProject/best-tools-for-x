## About
DigitalOcean provides VMs and managed services for app hosting. Spin up a server to run your stack with full control. Use it for projects needing raw cloud power over managed platforms.

## Features
- Launch VMs in under a minute
- Add block storage as needed
- Use prebuilt app images
- Monitor usage in dashboard
- SSH into servers directly
