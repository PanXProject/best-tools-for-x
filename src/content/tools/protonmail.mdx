## About
ProtonMail secures emails with end-to-end encryption. Use it to send sensitive messages without third-party access. Ideal for privacy-focused teams or individuals.

## Features
- Encrypt emails automatically
- Send self-destructing messages
- Use on web or mobile
- Import existing emails
- Add custom domains
