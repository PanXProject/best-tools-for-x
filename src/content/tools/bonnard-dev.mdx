## About
  Bonnard adds interactive, agent-ready charts to your MCP server in a few lines. The
  `@bonnard/mcp-charts` package renders charts inside Claude, ChatGPT, Cursor, and any
  MCP Apps host, generated from your real query data with no frontend work. Bonnard
  also offers a governed MCP platform: secure, multi-tenant MCP access to your
  customers' data over your warehouse, dbt, or semantic layer.

  ## Features
  - Add interactive charts to an MCP server with one function call
  - Renders natively in Claude, ChatGPT, Cursor, Gemini, and any MCP Apps host
  - Charts generated from real query data, not agent guesses
  - Governed, multi-tenant MCP access to customer data (auth, tenancy, RBAC)
  - Deploy managed, in your VPC, on-prem, or air-gapped
  - Connects Snowflake, BigQuery, Databricks, PostgreSQL, ClickHouse, DuckDB, Redshift
