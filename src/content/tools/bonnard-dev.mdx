## About
  Bonnard is an open-source semantic layer for AI agents and embedded
  analytics. Define business metrics in YAML, deploy as an API, and let AI
  agents query governed data via MCP or REST. Built-in multi-tenant row-level
  security and pre-aggregation for sub-second responses.

  ## Features
  - Define metrics in YAML and version in Git
  - AI agents query via MCP (Model Context Protocol)
  - Multi-tenant row-level security on every query
  - Pre-aggregation engine for sub-second responses
  - REST API and SDK for embedded analytics
  - Supports Snowflake, BigQuery, Databricks, PostgreSQL, ClickHouse, DuckDB,
  Redshift
