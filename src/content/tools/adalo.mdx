## About
Adalo enables anyone to build native mobile and web applications without coding. Create fully functional apps with databases, user authentication, and custom workflows using visual drag-and-drop interface.

## Features
- Visual app builder interface
- Native mobile app publishing
- Custom database creation
- User authentication systems
- Push notifications
- In-app purchases
- API integrations
- Progressive web app support
