## About
Magic Eraser is an AI image editing toolkit for object removal, background removal, enhancement, outpainting, generative fill, image generation, and design workflows.

## Features
- Remove objects, people, and text from photos
- Remove backgrounds with clean subject cutouts
- Improve sharpness, color, and photo detail with AI
- Extend image boundaries using generative AI
- Create social media graphics and marketing materials
