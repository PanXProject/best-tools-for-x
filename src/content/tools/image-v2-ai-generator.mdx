## About
Image 2 is an AI image generator and editor for teams creating ads, posters, ecommerce visuals, and social media images. It supports text-to-image generation, reference image editing, and model comparison in one workspace.

## Features
- Generate images from text prompts
- Edit with reference images
- Compare models in one workspace
- Create ad, poster, ecommerce, and social visuals
