## About
Coolors generates beautiful color palettes for designers and developers. Create, explore, and export color schemes with accessibility checking and format conversion for web and print projects.

## Features
- Random color palette generator
- Color harmony exploration
- Accessibility contrast checker
- Export in multiple formats
- Color blindness simulator
- Trending color collections
- Personal palette library
- Brand color extraction
