## About
Evidence builds data reports using SQL queries and Markdown. Host it yourself to create dashboards from your database. Use it for internal reporting where you control the stack and data.

## Features
- Write reports in Markdown
- Run SQL queries directly
- Self-host for private reports
- Embed charts from queries
- Share static HTML outputs
