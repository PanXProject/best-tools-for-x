## About
Moodle is a free, self-hosted platform for creating online courses. Use it to teach unlimited students with full control over content. Ideal for educators or businesses offering free training.

## Features
- Create unlimited courses
- Host on your own server
- Add quizzes and assignments
- Manage student progress
- Customize with plugins
