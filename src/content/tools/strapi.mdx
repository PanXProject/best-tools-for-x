## About
Strapi powers content for apps or sites, self-hosted for control. Use it to manage blog posts or product data via APIs. Ideal for devs building custom frontends with flexible backends.

## Features
- Create content types easily
- Self-host with Docker
- Generate REST or GraphQL APIs
- Manage users and roles
- Extend with custom plugins
