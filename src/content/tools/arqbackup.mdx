## About
Arq Backup secures files to cloud or disks with a one-time purchase. Use it to protect your desktop data without subscriptions. Ideal for users wanting long-term backup control.

## Features
- Backup to Dropbox or local
- Encrypt files automatically
- Restore old file versions
- Schedule hourly backups
- Verify data after backup
