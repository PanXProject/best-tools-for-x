## نبذة
أدي آي أو يحمي بريدك الإلكتروني الحقيقي من خلال إنشاء عناوين بديلة غير محدودة. كل خدمة تحصل على عنوان فريد لمنع الرسائل المزعجة والتتبع. حل مفتوح المصدر مع دعم الرد المجهول.

## المميزات
- إنشاء عناوين بريد إلكتروني بديلة
- ردود مجهولة لحماية الهوية
- خيار الاستضافة الذاتية للتحكم الكامل
- امتداد متصفح للإنشاء السريع
- خيارات نطاقات متعددة
- دعم تشفير PGP
- إحصائيات إعادة توجيه مفصلة
- وصول API للأتمتة
- وظائف استيراد/تصدير
- إدارة المستلمين المخصصة
- تتبع النطاق الترددي والتخزين
- خيارات تصفية متقدمة
- دعم تطبيق الجوال
- النسخ الاحتياطي والاستعادة
- واجهة متعددة اللغات
- إشعارات webhook
