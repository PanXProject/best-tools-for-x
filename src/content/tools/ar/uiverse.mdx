## نبذة
يوآي فيرس يوفر مكونات واجهة مستخدم ورسوم متحركة CSS جاهزة للاستخدام يمكن نسخها مباشرة إلى المشاريع. مكتبة مدفوعة بالمجتمع بتصاميم حديثة توفر وقت التطوير. جميع المكونات متجاوبة وقابلة للتخصيص.

## المميزات
- آلاف المكونات للنسخ واللصق
- رسوم متحركة وتأثيرات CSS خالصة
- معاينات تفاعلية للمكونات
- أنماط تصميم متجاوبة
- أدوات إنشاء مكونات مخصصة
- وظائف البحث والتصفية
- خيارات موضوع مظلم وفاتح
- نظام تصنيف المكونات
- ملف المستخدم والمجموعات
- وضع علامات مرجعية للمكونات المفضلة
- تحميل المكونات كملفات
- واجهة محسنة للجوال
- تحرير الكود في الوقت الفعلي
- ميزات المشاركة الاجتماعية
