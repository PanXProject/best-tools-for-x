## نبذة

إتش تي تي بي آي هو عميل HTTP سهل الاستخدام مصمم لاختبار وتصحيح والتفاعل مع واجهات برمجة التطبيقات وخوادم HTTP. يوفر واجهة سطر أوامر بديهية وتطبيق ويب يجعل طلبات HTTP مقروءة وسهلة التعامل معها.

## المميزات

- صيغة تعبيرية وبديهية
- دعم JSON مدمج مع تمييز الصيغة
- دعم واجهات REST و GraphQL البرمجية
- جلسات مستمرة وتوثيق
- نظام إضافات للتوسع
- أداة سطر أوامر متعددة المنصات
- واجهة ويب للاختبار في المتصفح
- قدرات التحميل والرفع
- رؤوس مخصصة وطرق الطلب
- تنسيق مخرجات جميل
