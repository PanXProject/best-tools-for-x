## نبذة

خدمة API زاحفة تحول موقع الويب إلى مخرجات منظمة مع تنظيف النصوص وتجهيزها للاستخدام في تدريب نماذج اللغة أو أرشفة المحتوى أو بناء محركات بحث داخلية.

## المميزات

- زحف جميع صفحات الموقع وإخراج محتوى منظم.
- تنظيف النصوص وإزالة الحشو لتجهيزها للـLLM.
- واجهة برمجية للاستخدام الآلي والتكامل.
- مناسب لبناء قواعد معرفة داخلية أو لتغذية وكلاء ذكاء اصطناعي.
