## نبذة

Dorar.net Hadith API هو واجهة برمجية وسيطة (Proxy API) تتيح البحث في الأحاديث من موقع الدرر السنية بدون قيود الـ CORS أو JSONP، وتعيد النتائج بهيئة JSON منظمة مع وثائق استخدام واضحة. توفر مزايا إضافية غير موجودة في الـ API الرسمي، مثل إزالة عناصر HTML ودعم البحث المتقدم مع التحكم في المعلمات.

## المميزات

* تجاوز قيود CORS وJSONP لموقع الدرر السنية
* إعادة البيانات بصيغة JSON منظمة وسهلة الاستخدام
* بحث في الأحاديث عبر API الرسمي أو عبر صفحة الموقع
* دعم البحث المتقدم باستخدام معلمات متعددة (rawi, book, mohdith, grade...)
* إزالة عناصر HTML من النصوص عند الحاجة
* دعم استرجاع الأحاديث المشابهة أو البديلة الصحيحة أو الأصول
* جلب شروح الأحاديث النصية والموسعة
* جلب معلومات عن المحدثين والكتب
* نظام Cache لتسريع الاستجابة وتخفيف الحمل على المصدر
* حدود استخدام يومية قابلة للتخصيص لكل IP
* توثيق كامل عبر Postman وSwagger UI
* دعم عرض نتائج مخصصة للمتخصصين أو غير المتخصصين
* إمكانية استبعاد كلمات أو جمل من البحث
* مرونة في نطاق البحث (أحاديث مرفوعة، قدسية، آثار الصحابة...)
* دعم البحث المطابق أو بجميع الكلمات أو بأي كلمة
* إمكانية التشغيل محليًا أو رفعه على الاستضافة الخاصة
