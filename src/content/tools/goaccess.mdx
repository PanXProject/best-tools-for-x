## About
GoAccess analyzes web server logs in real time, free and open-source. Self-host it to track site stats privately. Great for developers or small businesses wanting detailed, no-cost analytics.

## Features
- Analyze logs in real time
- Generate HTML or terminal reports
- Track visits, referrers, and more
- Run on your own server
- No usage limits
