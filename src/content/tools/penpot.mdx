## About
Penpot is a free, open-source alternative for UI design collaboration. Host it yourself to keep projects private. Use it for team design workflows without subscription fees or vendor lock-in.

## Features
- Collaborate on designs in real time
- Export SVG files directly
- Self-host for data control
- Use built-in prototyping tools
- Access open-source community plugins
