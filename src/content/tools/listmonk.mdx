## About
Listmonk sends newsletters from your server with no subscription fees. Use it to manage email lists and campaigns privately. Ideal for creators or orgs needing full control over their mail stack.

## Features
- Send bulk emails from your server
- Import subscribers via CSV
- Create reusable templates
- Track opens and clicks
- Self-host with minimal setup
