## About
Cloudflare speeds up sites with a CDN and blocks attacks. Use it to protect and optimize your web presence. Great for devs or businesses running public-facing sites.

## Features
- Cache content globally
- Block DDoS attacks
- Add SSL with one click
- Monitor traffic stats
- Route DNS securely
