## About
Excalidraw is an interactive whiteboard tool enabling real-time collaborative sketching with a natural, hand-drawn feel.

## Features
- Real-time collaboration  
- Hand-drawn style diagrams  
- Shape library and free drawing  
- Export as PNG or SVG  
- Offline support  
- Version history
