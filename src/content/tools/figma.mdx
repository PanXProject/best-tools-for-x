## About
Figma lets design teams work on UI prototypes together in real time. Share files instantly and get feedback without exports. Perfect for remote teams iterating on app or web designs quickly.

## Features
- Edit designs with teammates live
- Comment directly on prototypes
- Export assets in multiple formats
- Use shared component libraries
- Access version history instantly
