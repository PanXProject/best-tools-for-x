## About
Wave offers free accounting software tailored for small businesses and freelancers. It simplifies invoicing, expense tracking, and financial reporting, making it ideal for bootstrappers managing finances without a dedicated accountant.

## Features
- Create and send professional invoices
- Track expenses with bank account integration
- Generate financial reports
- Free with no hidden fees
- User-friendly web interface
